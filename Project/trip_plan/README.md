환경설정
    1. 파이썬 v.3.11 설치
    2. 환경설정(파이썬 v.3.11 설치 경로)
    3. cmd창에서
        pip install poetry
    4. vscode trip_plan안에
        .env 설정(API KEY)
    5. vscode terminal 열고(command prompt로 연다)
    6. poetry install
    7. 위 6번에서 생성한 가상환경으로 kernel을 변경한다.(vscode 우측상단)

실행방법
    환경 설정이 끝나면 코드 창을 하나씩 실행한다.
